# quotes-parser
Консольный парсер цитат с quotes.toscrape.com на Python. Обходит все страницы сайта, собирает цитаты, авторов и теги, экспортирует в CSV или JSON, умеет группировать цитаты по тегам. Обрабатывает сетевые ошибки, не падает при недоступности сайта. Стек: requests, BeautifulSoup4.


# Quotes Parser

Консольный парсер цитат с сайта [quotes.toscrape.com](https://quotes.toscrape.com/) на Python.

Обходит все страницы сайта, собирает цитаты вместе с авторами и тегами, экспортирует результат в CSV или JSON, умеет группировать собранные цитаты по тегам.

## Возможности

- Постраничный обход всего сайта (автоматическая остановка, когда цитаты заканчиваются)
- Экспорт в CSV или JSON — на выбор через интерактивное меню
- Группировка и вывод цитат по тегам
- Обработка сетевых ошибок (таймауты, недоступность сайта) без падения программы
- Задержка между запросами, чтобы не перегружать сайт

## Стек

- Python 3
- [requests](https://pypi.org/project/requests/) — HTTP-запросы
- [beautifulsoup4](https://pypi.org/project/beautifulsoup4/) — разбор HTML

## Установка

```bash
git clone https://github.com/<твой-username>/quotes-parser.git
cd quotes-parser
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

Скрипт соберёт все цитаты с сайта, после чего предложит:

1. Выбрать формат экспорта — CSV или JSON
2. Опционально сгруппировать и вывести цитаты по тегам

## Пример вывода

```
Парсим. Страница 1
Страница 2 пустая, цитаты закончились
Всего собрано: 100 цитат

Хотите ли вы экспортировать цитаты?
Y/N
```

Результат сохраняется в `quotes_csv.csv` или `quotes.json` в корне проекта.

## Формат CSV

| Цитата | Автор | Теги |
|---|---|---|
